# School21 Telegram-бот

Telegram-бот для работы с [открытым API School21](https://platform.21-school.ru): профиль, проекты, события кампуса — и автоматические уведомления, когда проект уходит в проверки или в кампусе появляется новое событие.

## Возможности

- **Меню на инлайн-кнопках** (`/start`): профиль, поинты, скиллы, логтайм, фидбек, история XP, бейджи, коалиция, рабочее место, курсы, проекты по статусам, кампусы (участники, коалиции, кластеры, карты), события, продажи, граф проектов.
- **Автослежка** (фоновый поток): каждые `POLL_INTERVAL_SECONDS` секунд бот проверяет статусы проектов и события кампуса и присылает уведомление при изменениях — например, «🚨 УШЁЛ В ПРОВЕРКИ!» или «🆕 Новое событие в кампусе».
- **ПИНГ-кнопки**: ручная проверка новых проверок и событий на неделю вперёд.
- **Бейджи с иконками**: картинки скачиваются с платформы (нужен токен, напрямую Telegram их не видит) и приходят одним альбомом с одной кнопкой удаления.
- **Слежка за любым участником**: `/track ник` — переключиться на другого (ему ничего делать не нужно), `/track` — вернуть свой. Выбор сохраняется в `state.json` навсегда.
- Сообщения с результатами удаляются кнопкой 🗑, меню всегда редактируется на месте.

## Состав проекта

| Файл | Назначение |
|---|---|
| `bot.py` | Основной бот: меню, действия, автослежка |
| `main.py` | Упрощённый вариант: только поллинг статусов проектов с уведомлениями |
| `school21_client.py` | Минимальный клиент School21 API (авторизация через Keycloak, GET-запросы) |
| `telegram_notifier.py` | Отправка сообщений в Telegram (используется `main.py`) |
| `sberapi.txt` | Справка по эндпоинтам School21 OpenAPI |

## Установка

```bash
git clone <url-репозитория>
cd <папка>
python3 -m venv venv
./venv/bin/pip install -r requirements.txt
```

## Настройка

Скопируй `.env.example` в `.env` и заполни:

```bash
cp .env.example .env
```

| Переменная | Описание |
|---|---|
| `S21_LOGIN` | Логин на платформе School21 |
| `S21_PASSWORD` | Пароль от платформы |
| `S21_TRACK_LOGIN` | Чей прогресс отслеживать (по умолчанию — свой) |
| `TG_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `TG_CHAT_ID` | ID чата, куда слать сообщения |
| `POLL_INTERVAL_SECONDS` | Интервал автослежки в секундах (по умолчанию 120) |

`.env` в git не попадает — он в `.gitignore`.

## Запуск

Полноценный бот с меню и автослежкой:

```bash
./venv/bin/python bot.py
```

Или только уведомления о смене статусов проектов, без меню:

```bash
./venv/bin/python main.py
```

После запуска напиши боту `/start` — он пришлёт главное меню.

## Как это работает

1. `school21_client.py` получает JWT-токен у Keycloak (`auth.21-school.ru`) по логину и паролю и автоматически обновляет его по истечении срока.
2. Бот работает через long polling Telegram (`getUpdates`), без вебхуков — можно запускать где угодно.
3. Состояние (какие проекты уже в проверках, какие события уже показаны) хранится в `state.json` рядом со скриптом, поэтому перезапуск не вызывает повторных уведомлений.
